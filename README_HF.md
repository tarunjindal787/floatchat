# FloatChat 🌊

AI-Powered Conversational Interface for ARGO Ocean Data Discovery and Visualization.

**Features:**
- Natural language queries for ARGO float data
- Geospatial trajectory maps & profile visualizations
- BGC parameter dashboards (O₂, Chl-a, NO₃)
- T-S diagrams, Hovmöller sections
- Data export (CSV, NetCDF, ASCII)

**Source:** [github.com/tarunjindal787/floatchat](https://github.com/tarunjindal787/floatchat)
